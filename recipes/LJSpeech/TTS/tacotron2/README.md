# LibriSpeech ASR with seq2seq models (CTC + attention).
This folder contains the scripts to train a Tacotron Text-to-Speech system using the LJSpeech dataset.

You can download LJSpeech at http://www.openslr.org/12

# How to run
python train.py hparams/hparams.yaml


# Training Time
It takes around 7 minutes per epoch to train Tacotron

# PreTrained Model + Easy-Inference
A downloadable pretrained model will be made available in a future release
